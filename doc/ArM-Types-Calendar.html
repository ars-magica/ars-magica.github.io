Calendar

The software assumes the Hibernian calendar, with Winter being the first season of the year. Several things may have to change for the standard calendar with Winter as the last season of the year. We have tried to collect these definitions here.

Hibernian Calendar

This comparison is used to check if a character is past the age limit where aging rolls are required. ``` (*>) = (>=) ```
