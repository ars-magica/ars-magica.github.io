data Advancement #

The advancement object has two roles. It can hold the advancemet from one season or chargen stage, as specified by the user. It can also hold additional field inferred by virtues and flaws. One may consider splitting these two functions into two types.

Advance Character #

The implementation of character advancement depends on three auxiliary functions. + prepareCharacter advances the character from a Nothing state to Game Start. + prepareAdvancement augments the advancement object with limits and inference + applyAdvancement applies an advancement to advance the character a single step.

Additional inference should be added to one of these functions. + prepareCharacter (see below) if it applies to Character Generation only + prepareAdvancement if it modifies the advancement only + applyAdvancement if it modifies the CharacterState
