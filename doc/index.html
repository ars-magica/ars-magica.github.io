harm

The harm module provides a CLI interface to process Ars Magica characters from JSON files and produce character sheets in Markdown.

The JSON file normally contains a list of advancement stages, and the actual state of the character at a given time is computed.

The libraries are designed with the intention of creating client/server systems with different interactie and non-interactive clients.
