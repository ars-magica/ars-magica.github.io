type XPType = Float #

The XPType is the taype of XP. By RAW it should be Int, but some troupes prefer to count fractional XP when affinities are involved. This requires changing to a floating point type.

Calculate the XP cost for a given score on the pyramid scale

class GenericXPType a where #

The class provides functions which must be implemented differently depending on the definition of XPType.
